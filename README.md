O **Projeto Loja de Ferragem** é um sistema educacional desenvolvido em **TypeScript**, com foco em **Programação Orientada a Objetos (POO)** e **arquitetura modular**, simulando a gestão de produtos e operações de uma loja.

**Objetivo:** Demonstrar organização de código, domínio técnico e aplicação de boas práticas de engenharia de software em um projeto de portfólio.

<br />

> ⚠️ Este projeto possui fins educacionais e não representa um sistema comercial real.

---

## 🎯 Competências Técnicas Demonstradas

- Programação Orientada a Objetos (Encapsulamento, Herança, Polimorfismo)
- Modelagem de domínio
- Arquitetura em camadas (Model, Repository, Controller)
- Tipagem forte com TypeScript
- Separação de responsabilidades
- Organização modular
- Validação de dados
- Controle de fluxo
- Estrutura preparada para evolução futura (API, banco de dados, testes)

## 🏗 Arquitetura do Projeto

Estrutura organizada para facilitar**manutenção e escalabilidade**:

```text
📦 LojaDeFerragem
┣ 📂 src
┃ ┣ 📂 controller
┃ ┣ 📂 model
┃ ┣ 📂 repository
┃ ┗ 📂 util
┣ 📜 Menu.ts
┗ 📜 tsconfig.json
```


### 📌 Organização

- **Model** → Entidades de domínio
- **Repository** → Simulação de persistência
- **Controller** → Regras de negócio
- **Menu.ts** → Ponto de entrada da aplicação

## 🚀 Como Executar

## Como Executar



**1️⃣ Clone o repositório**

```bash
git clone https://github.com/LeoIbanhez92/LojaDeFerragem.git
```

**2️⃣ Acesse a pasta do projeto via terminal**

```bash
cd LojaDeFerragem
```

**3️⃣ Instale as dependências**

```bash
npm install
```

**4️⃣ Execute a aplicação**

```bash
ts-node Menu.ts
```
